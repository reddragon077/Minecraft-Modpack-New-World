# New World - Proje Tasarım Belgesi

## Vizyonumuz

Minecraft'ta tamamen farklı bir oynanış deneyimi yaratmak: Keşif-tabanlı, araştırma-odaklı, hikâye-gerçekçi.

## Temel Mekanikler

### 1. Araştırma Sistemi

Oyuncu hiçbir teknolojiyi başlangıçta bilmez. Teknolojiler şu yollarla açılır:
- Araştırma laboratuvarları
- Veri diskleri
- Eski tesisler
- Hafıza kayıtları
- Keşif görevleri

### 2. Bilgi Merkezli Kaynaklar

Kaynak sistemi:
- **Geleneksel:** Demir, Bakır, vb madde kazı → Yapı → Üretim
- **Yeni World:** Element keşfi → Analiz → Veri tabanı → Replicator

### 3. Modüler Fabrika

Production Chamber sistemi:
- Oyuncu hazır makine craft etmez
- Boş üretim odası oluşturur
- İçeride kendi fabrikasını tasarlar
- Dış dünyada tek blok görünür

### 4. Gemi Geliştirmesi

Ana merkez: Uzay gemisi
Ilerledikçe açılan bölümler:
- Laboratuvarlar
- Reaktör odası
- Biyoloji laboratuvarı
- Araştırma merkezi
- Portal odaları
- Otomasyon odaları

### 5. Genetik Sistem

İksir yerine DNA geliştirmesi:
- Night Vision → Göz geliştirmesi
- Haste → Kas optimizasyonu
- Fire Resistance → Isı dayanıklılığı
- Vb.
